# C语言基础语法

参照07_Materials/01_Book中的书籍，提取出基本语法的代码示例，能够快速的熟悉C语言的基本内容。

- 1 《C程序设计语言_第2版》
非常精练的描述了C语言的基础语法，适合作为入门学习。1988年出版，基本覆盖了C89。

- 2 《C语言程序设计_现代方法_第2版修订版》
完整的描述了C语言语法。2021年出版，从C89一直到C18。

先学习这里的基础概念和示例代码，后续再细看《C语言程序设计_现代方法_第2版修订版》更完整的学习C语言

# Quick start

- 抄写与运行示例代码，可达到对C语言的快速入门
