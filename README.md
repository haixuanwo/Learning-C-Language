# 二、C语言介绍
- 编程语言：是人们用来“指挥”计算机做事的介质，是人与计算机之间沟通的语言。
- 软件：通过编程语言让计算机实现完成用户想要的功能系统
- 软件示例：计算器、文字处理软件、视频播放器、游戏、数据库管理系统、网络浏览器、操作系统、数据库服务器、打印机驱动程序、打印服务器、路由器、智能手机、智能手表、智能电视、智能音箱、智能机器人、智能家居、智能家电、智能车、智能医疗、智能健康、智能冰箱、智能客厅、智能厨房、智能卫浴、智能客服、智能电动车、智能电视机、智能电视盒、智能电视机柜、智能电视遥控器、智能电视盒子、智能电视机顶盒、智能电视机架、智能电视机

- C语言：使用C语言来描绘出一个逻辑系统，通过输入让软件系统产生输出，实现用户的需求。

如计算器，程序输入需计算的数据和方法
输入：1+2*3-4/5
程序处理：
    int a=1,b=2,c=3,d=4,e=5;
    float result=(float)(a+b*c-d/e);
    return result;
输出：-0.6
